# vega-desktop

一个运行在浏览器中的仅实现了部分UI样式组件的精简版的`操作系统`，基于Vue3.0实现

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
